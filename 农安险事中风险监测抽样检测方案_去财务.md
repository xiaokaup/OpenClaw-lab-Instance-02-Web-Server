## 1. 项目概述

### 1.1 项目名称

农安险投保主体事中风险监测抽样检测项目

### 1.2 项目目标

通过线下人工抽样检测，贯穿农产品生产全过程，实现事中风险监测与预警，降低保险赔付风险，保障农产品质量安全。

### 1.3 覆盖品类

- **品类一**：蔬菜（绿叶菜为代表，生长期约35天）
- **品类二**：生猪（育肥期约150天）
- **品类三**：淡水鱼（草鱼为代表，养殖期约8-10个月）
- **品类四**：水果（荔枝为代表，果实发育期约70天）

### 1.4 设计原则

- **成本优先**：不使用昂贵仪器，以感官检测+快速检测卡（胶体金/酶抑制）为主
- **全程覆盖**：产前→产中→采收前→上市前，四个关键节点
- **风险导向**：高风险品类/主体提高频次，低风险降低频次
- **数据可追溯**：每次抽样有记录，可回查

### 1.5 实施模式

线下人工巡回抽样 + 胶体金/酶抑制快速检测 + 可疑样品第三方定量验证

### 1.6 预期规模（首期试点）

覆盖50户投保主体，分布于5个乡镇。

## 2. 抽样检测方案

### 2.1 通用抽样节点

| 节点 | 内容 | 方式 |
| ------ | ------ | ------ |
| 产前 | 投入品把关（种子/农药/肥料来源核查） | 台账核对 |
| 产前 | 土壤/水质（仅高风险地块） | 速测卡 |
| 产中 | 多次生长过程农残速测 | 胶体金卡 |
| 产中 | 外观异常巡查 | 感官检查 |
| 采收前 | 关键关口综合检测 | 酶抑制法+胶体金多联检 |
| 上市前 | 出库把关快检 | 速测卡/酶抑制法 |
| 上市前 | 抽样送检（5-10%比例） | 委托三方实验室 |

### 2.2 风险分级与抽样频次

| 风险等级 | 定义 | 抽样频次 |
| ---------- | ------ | ---------- |
| 🔴 高风险 | 易超标品类（绿叶菜、豇豆、韭菜等） | 每茬必检，出库批批检 |
| 🟡 中风险 | 常规品类（茄果类、瓜类） | 每茬抽1次+采收前 |
| 🟢 低风险 | 耐储/少用药品类（根茎类、食用菌） | 每茬抽1次或免检 |

### 2.3 低成本工具清单

- **胶体金速测卡** — 针对特定农药，现场15分钟出结果
- **便携式农残速测仪** — 酶抑制率法
- **智能记录APP（微信小程序级）** — 拍照+定位+填表，替代纸质台账
- **感官评估表** — 标准化打分表

### 2.4 预警触发机制

| 预警级别 | 触发条件 | 处置动作 |
|----------|----------|----------|
| 🟢 一级（提示） | 速测阳性但非禁限用药，数值接近限值 | 通知主体加强管理，取样复查 |
| 🟡 二级（警告） | 速测明确阳性，或记录缺失/造假 | 暂停出库，送检定量，约谈主体 |
| 🔴 三级（严重） | 检出禁限用药，或连续两次不合格 | 启动赔付反向核验，标记黑名单 |

## 2.5 蔬菜（绿叶菜）产中抽样方案·以天为单位

以小白菜为例，生长期约35天。

### 阶段一：苗期（第1—10天）｜核心指标：生长基础

| 天数 | 抽样内容 | 检测指标 | 方法 |
| ------ | ---------- | ---------- | ------ |
| 第1天 | 土壤/基质 | 土壤湿度、pH值、EC值 | 便携三合一测量笔 |
| 第1天 | 灌溉水 | pH、电导率、肉眼可见杂质 | pH试纸+感官 |
| 第3天 | 出苗率 | 出苗率（随机5点取样，每点1㎡） | 人工计数 |
| 第5天 | 幼苗外观 | 叶色（是否黄化）、株高均匀度、有无根腐 | 感官评估+尺量 |
| 第7天 | 投入品 | 追肥/用药记录核实 | 台账核对，检查是否用禁限药 |
| 第10天 | 虫害初筛 | 蚜虫、跳甲基数（黄色粘虫板抽检） | 人工计数+放大镜 |

> 第10天汇总：不达标地块标记一级预警。

### 阶段二：快速生长期（第11—25天）｜核心指标：用药与残留

| 天数 | 抽样内容 | 检测指标 | 方法 |
| ------ | ---------- | ---------- | ------ |
| 第12天 | 叶片/茎秆 | 有机磷类（毒死蜱、乐果、氧化乐果） | 酶抑制速测卡（3合1） |
| 第14天 | 叶片/茎秆 | 氨基甲酸酯类（克百威、涕灭威） | 胶体金速测卡 |
| 第16天 | 全株 | 菊酯类（氯氰菊酯、高效氯氟氰菊酯） | 胶体金速测卡 |
| 第18天 | 叶片 | 烟碱类（吡虫啉、啶虫脒） | 胶体金速测卡 |
| 第20天 | 叶片/茎秆 | 综合速测：多合一胶体金联检卡（有机磷+菊酯+烟碱，5项） | 胶体金多联检卡 |
| 第22天 | 叶片 | 杀菌剂类（多菌灵、百菌清） | 胶体金速测卡 |
| 第25天 | 全地块 | 阶段评估：综合农残+虫害密度+生长状态 | 速测+感官+台账 |

### 阶段三：成熟期/采收前（第26—35天）｜核心指标：最终合规

| 天数 | 抽样内容 | 检测指标 | 方法 |
| ------ | ---------- | ---------- | ------ |
| 第26天 | 全株 | 安全间隔期核验（最后用药日+品种间隔期） | 台账核对 |
| 第28天 | 叶片/茎秆 | 全面农残速测：多联检卡全覆盖 | 胶体金8项联检卡 |
| 第30天 | 叶片/茎秆 | 酶抑制率定量：抑制率%值 | 便携速测仪，10个采样点混合样 |
| 第32天 | 成品样 | 感官终评：外观、色泽、新鲜度、有无病斑虫眼 | 评定表，拍照留档 |
| 第34天 | 成品样 | 抽样留样（高风险批次） | 封袋标记，-18℃留存7天 |
| 第35天 | 出库批样 | 批批快检（高风险必检；中低风险按20%抽） | 胶体金联检卡 |

### 蔬菜检测指标汇总

| 指标大类 | 具体项目 | 检测节点 | 方法 | 限值依据 |
|----------|----------|----------|------|----------|
| 有机磷类 | 毒死蜱、乐果、氧化乐果、甲胺磷 | 第12/20/28天 | 酶抑制/胶体金 | GB 2763 |
| 氨基甲酸酯类 | 克百威、涕灭威 | 第14/28天 | 胶体金 | GB 2763 |
| 菊酯类 | 氯氰菊酯、高效氯氟氰菊酯 | 第16/28天 | 胶体金 | GB 2763 |
| 烟碱类 | 吡虫啉、啶虫脒 | 第18/28天 | 胶体金 | GB 2763 |
| 杀菌剂类 | 多菌灵、百菌清 | 第22/28天 | 胶体金 | GB 2763 |
| 综合抑制率 | 酶抑制率（总有机磷指示） | 第30天 | 速测仪 | 抑制率≥50%=阳性 |
| 生长指标 | 出苗率、株高、叶色、虫害密度 | 第3/5/10/25天 | 人工 | 品种标准 |
| 台账合规 | 用药记录、安全间隔期 | 第7/26天 | 核对 | 农药标签 |

## 2.6 生猪产中抽样方案·以天为单位

育肥周期约150-180天。

### 阶段A：仔猪入栏期（第1—7天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 第1天 | 仔猪来源核查 | 检疫证明、耳标号、来源场备案 | 台账核对 |
| 第1天 | 猪舍环境 | 温度、湿度、氨气浓度 | 便携温湿度计+氨气试纸 |
| 第3天 | 饮水 | pH、大肠菌群 | 试纸+快速检测片 |
| 第5天 | 饲料 | 黄曲霉毒素B1、呕吐毒素 | 胶体金速测卡 |
| 第7天 | 猪只健康 | 精神状态、呼吸、粪便、皮肤（逐头感官） | 感官评估+台账 |

### 阶段B：育肥前期（第30、60天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 第30天 | 尿样（抽10%存栏或至少5头） | 瘦肉精三联（克伦特罗、莱克多巴胺、沙丁胺醇） | 胶体金三联卡 |
| 第30天 | 饲料 | 喹乙醇、土霉素等抗生素 | 胶体金速测卡 |
| 第60天 | 尿样（抽10%） | 瘦肉精+四环素类+磺胺类 | 胶体金多联检卡 |
| 第60天 | 饲料 | 霉菌毒素复查 | 胶体金卡 |

### 阶段C：育肥后期/出栏前（第90、120、出栏前7天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 第90天 | 尿样+饲料 | 瘦肉精+氟喹诺酮类 | 胶体金卡 |
| 第120天 | 尿样+饮水 | 瘦肉精+氯霉素 | 胶体金卡 |
| 出栏前7天 | 尿样（必检，全群10%） | 瘦肉精+抗生素8项联检 | 胶体金多联检卡 |
| 出栏前3天 | 抽血样（高风险场） | 非洲猪瘟抗原快速检测 | 胶体金/荧光卡 |
| 出栏日 | 尿样（批批检） | 瘦肉精必检 | 胶体金卡 |

### 生猪关键指标

- 🚨 瘦肉精类（克伦特罗/莱克多巴胺/沙丁胺醇）：每30天+出栏必检
- 🚨 抗生素类（四环素/磺胺/氟喹诺酮/氯霉素）：第60/90/120天
- 🚨 饲料毒素（黄曲霉B1/呕吐毒素）：入栏+第60天
- 🚨 疫病（非洲猪瘟）：高风险场出栏前3天
- 🚨 环境与饮水：入栏期

## 2.7 淡水鱼（草鱼）产中抽样方案·关键节点

养殖期约8-12个月。

### 阶段A：放苗前后（第1—7天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 第1天 | 池塘水质 | pH、氨氮、亚硝酸盐、溶氧 | 便携五合一水质检测盒 |
| 第1天 | 鱼苗 | 来源证明、检疫证 | 台账核查 |
| 第3天 | 底泥（老塘必检） | 孔雀石绿、重金属铅镉 | 胶体金速测卡 |
| 第7天 | 鱼苗存活率 | 随机3点撒网计数 | 人工计数 |

### 阶段B：养成期（第30、90、150、210天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 第30天 | 水质 | pH、氨氮、亚硝酸盐 | 检测盒 |
| 第30天 | 饲料 | 喹乙醇、黄曲霉毒素 | 胶体金卡 |
| 第90天 | 鱼体（捞3-5尾） | 孔雀石绿+结晶紫（隐性/显性） | 胶体金双联卡 |
| 第90天 | 鱼体 | 硝基呋喃类代谢物（呋喃西林/呋喃唑酮） | 胶体金卡 |
| 第150天 | 鱼体 | 氯霉素+氟喹诺酮类 | 胶体金多联检卡 |
| 第210天 | 鱼体 | 孔雀石绿+硝基呋喃+磺胺类 | 多联检卡 |

### 阶段C：上市前（出塘前14天、7天、出塘日）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 出塘前14天 | 鱼体+水质 | 孔雀石绿+氯霉素+硝基呋喃+氟喹诺酮 | 胶体金多联检卡 |
| 出塘前7天 | 鱼体（全塘抽样5-10尾） | 禁限用药全面筛查 | 胶体金8项联检 |
| 出塘日 | 批批快检 | 孔雀石绿（必检）+氯霉素 | 胶体金双联卡 |

### 淡水鱼关键指标

- 🚨 孔雀石绿：底泥→养成期→上市前全程监控（水产头号禁用物）
- 🚨 硝基呋喃类：第90天+上市前
- 🚨 氯霉素/氟喹诺酮：第150天+上市前
- 🚨 水质（氨氮/亚硝酸盐）：关联用药风险，持续监控
- 🚨 饲料毒素：养殖高峰期检测

## 2.8 荔枝产中抽样方案·以天为单位

果实发育期约60-70天，以花后计。

### 阶段A：花期/幼果期（花后第1—20天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 花后第3天 | 花穗/幼果 | 有机磷类（毒死蜱、乐果） | 酶抑制速测卡 |
| 花后第7天 | 叶片+幼果 | 菊酯类（高效氯氟氰菊酯） | 胶体金卡 |
| 花后第10天 | 投入品 | 保花保果药剂台账核查 | 台账核对 |
| 花后第15天 | 叶片+幼果 | 杀菌剂（多菌灵、代森锰锌） | 胶体金卡 |
| 花后第20天 | 全株+土壤 | 毒死蜱+吡虫啉 | 胶体金联检卡 |

### 阶段B：果实膨大期（花后第25—55天）｜用药高峰

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 花后第25天 | 果实 | 有机磷+菊酯+烟碱类（多联检） | 胶体金多联检卡 |
| 花后第30天 | 果实 | 杀菌剂（咪鲜胺、嘧菌酯） | 胶体金卡 |
| 花后第35天 | 果实+叶片 | 毒死蜱+高效氯氟氰菊酯（荔枝最常用） | 胶体金卡 |
| 花后第40天 | 果实 | 杀虫剂+杀菌剂全覆盖 | 多联检卡 |
| 花后第45天 | 果实 | 昆虫生长调节剂（除虫脲等） | 胶体金卡 |
| 花后第50天 | 果实 | 安全间隔期核验 | 台账核对 |
| 花后第55天 | 果实 | 采收前全面速测（8项联检） | 胶体金多联检卡 |

### 阶段C：采收期（花后第58—65天）

| 天数 | 检测内容 | 指标 | 方法 |
| ------ | ---------- | ------ | ------ |
| 采收前3天 | 果实（随机5棵树，每树4方向各2颗） | 必检：毒死蜱+高效氯氟氰菊酯+咪鲜胺+吡虫啉 | 胶体金多联检卡 |
| 采收日 | 出园批检 | 有机磷+菊酯快速筛查 | 酶抑制+胶体金 |
| 采收后24h | 鲜果保鲜剂 | 二氧化硫/亚硫酸盐（熏硫残留） | 快速检测管 |

### 荔枝关键指标

- 🎯 毒死蜱：荔枝超标第一名，花后第3天起持续监控
- 🎯 高效氯氟氰菊酯（功夫菊酯）：膨大期重点
- 🎯 咪鲜胺/多菌灵：保鲜防腐，采收前必检
- 🎯 吡虫啉：花期+膨大期
- 🎯 二氧化硫：采后保鲜剂，测残留

## 3. 项目实施工作方案

### 3.1 组织架构

```
                项目领导小组
              （农安险管理方 + 农业部门代表）
                      │
              ┌───────┴───────┐
         项目负责人（1人）   技术顾问（1人）
              │
    ┌─────────┼─────────┐
  外勤组    实验室组    数据组
 (4-6人)   （1-2人）   （1人）
```

### 3.2 岗位职责

**项目负责人（1人，全职）**
- 制定年度/季度抽样计划
- 协调外勤排期与路线
- 审批检测报告，发布预警
- 对接保险理赔与农业主管部门
- 人员考核与管理

**技术顾问（1人，可兼职）**
- 负责检测方法的技术指导与培训
- 审核检测操作规程
- 处理检测争议与复检
- 跟踪法规标准更新（GB 2763等）

**外勤组（4-6人）**
- 组内分工：蔬菜1-2人、生猪1人、水产1人、水果1-2人（按片区+品类交叉覆盖）
- 按计划入户抽样、现场快速检测
- 填报抽样记录
- 可疑样品采样送检
- 现场感官评估与图片留档

**实验室组（1-2人）**
- 酶抑制率法速测仪操作与读数
- 胶体金卡检测结果判读
- 样品接收、编号、留样管理
- 第三方送检协调与报告归档

**数据组（1人）**
- 抽样检测数据录入与汇总
- 维持抽样台账与主体档案
- 对接系统预警模块，录入预警信息
- 编制周报/月报/季报

### 3.3 总编制：7-10人

**全职人员（5-6人）**
- 项目负责人 1人
- 外勤骨干 2人（各带1名兼职/临聘，共4人）
- 数据管理员 1人

**兼职/外包人员（2-4人）**

## 4. SOP 操作手册

### SOP-001 蔬菜（绿叶菜）抽样检测操作规程

#### 适用范围
小白菜、菠菜、生菜、油菜、空心菜等速生绿叶菜，生长期约30-45天。

#### 取样时机
苗期第3/5/10天、生长期第12/14/16/18/20/22/25天、成熟期第26/28/30/32天、采收日第35天。

#### 取样点位
以地块对角线5点取样法为准，每点采集全株2-3棵（根部不去除，土抖净装入样品袋）。5点取样覆盖地块四角和中心，每点间距不小于5米。避开地头地边1米区域。

#### 取样量
- 速测：5点混合后取代表性样品200g（约4-6株）
- 留样：另取相同量封装标记，-18℃保存7天

#### 取样时间
上午8:00-10:00，叶片无露水时。避开用药后24小时内。

#### 操作步骤

**步骤一：现场前处理**
- 样品不冲洗（模拟消费者状态）
- 用食品剪刀将叶片和茎秆剪碎至1cm以下
- 称取10g碎样于提取瓶中

**步骤二：提取**
- 加入配套提取液20mL
- 剧烈振荡2分钟（或手摇3分钟）
- 静置1分钟取上清液

**步骤三：检测**

**酶抑制法**：取上清液100μL加入酶试剂→37℃水浴15min→加底物→显色3min→速测仪读数。抑制率≥50%判阳性。

**胶体金法**：取上清液3-5滴入加样孔→水平放置15min→目视判读或读卡器读数。C线+T线均显色=阴性，仅C线显色=阳性，C线不显=无效。

**步骤四：结果记录**
- 阳性样品立即拍照→上传→通知项目负责人→启动复核程序
- 阴性结果正常录入台账

#### 注意事项
- 胶体金卡开封后立即使用，受潮失效
- 不同项目用不同卡，不可混用
- 酶抑制法每批次必须做空白对照
- 工作温度低于10℃需用保温袋预热试剂
- 取样手套每户一换，防止交叉污染

#### 阳性复核流程
- 胶体金阳性→同一批次样品用酶抑制法复测
- 酶抑制阳性→封样送第三方LC-MS/MS定量
- 第三方阳性→启动二级/三级预警处置

### SOP-002 生猪抽样检测操作规程

#### 适用范围
育肥生猪，体重30kg至出栏（约110-120kg）。

#### 取样时机
仔猪入栏第1/5/7天、育肥第30/60/90/120天、出栏前7天、出栏前3天（高风险场）、出栏日。

#### 尿样采集
- 抽样比例：存栏数10%，最少5头，最多20头
- 随机抽样：按耳标号尾数随机（如尾号3、7者）
- 采集方法：晨起首次排尿最佳，用一次性接尿杯接取中段尿≥50mL
- 猪只保定：由养殖户协助，确保人员安全和猪只福利

#### 饲料采样
- 料槽中不同位置取5份，混合后四分法缩至约200g
- 注意查看料槽是否有药物拌料痕迹（颜色不均）

#### 血样采集（仅高风险场，由兽医执行）
- 前腔静脉或耳缘静脉采血2mL
- 加抗凝剂，冷链保存，4小时内检测

#### 操作步骤

**步骤一：尿样前处理**
- 取尿液约3mL于离心管
- 如浑浊或可见杂质，3000rpm离心3min或静置5min取上清

**步骤二：胶体金卡检测（瘦肉精三联）**
- 恢复至室温后，滴管取3-4滴尿液加入加样孔
- 计时器计时5-8min（以说明书为准），不可超时
- C线+T线均显色=阴性，仅C线显色=阳性

**步骤三：饲料毒素检测**
- 饲料粉碎过40目筛（或便携粉碎机打粉）
- 称取5g样品+提取液25mL→振荡10min→离心/过滤→取上清→胶体金卡检测

**步骤四：非洲猪瘟抗原检测（高风险场）**
- 用无菌棉签蘸取抗凝血
- 加入样品稀释液混匀
- 滴入检测卡，15min判读

#### 注意事项
- 生猪采样必须有养殖户在场，穿戴防护服和手套
- 取样后场地和工具消毒（75%酒精或次氯酸钠）
- 尿样在常温下2小时内检测，超时冷藏（2-8℃）保存不超过24小时
- 非洲猪瘟检测呈阳性立即报告当地畜牧兽医部门
- 跨场作业必须更换防护服和手套

#### 生物安全防护
- 进场前：喷雾消毒通道或脚踏消毒垫
- 作业中：一次性防护服+手套+口罩+鞋套
- 出场后：防护服等废弃物装入医疗垃圾袋，交由无害化处理

### SOP-003 淡水鱼抽样检测操作规程

#### 适用范围
草鱼、鲢鱼、鳙鱼、鲤鱼、鲫鱼等淡水养殖经济鱼类。

#### 取样时机
放苗第1/3/7天、养成第30/90/150/210天、出塘前14天、出塘前7天、出塘日。

#### 水样采集
- 池塘四角和中心共5点，水面下50cm取水
- 每点约200mL，混合后取500mL为检测水样
- 采样瓶用池塘水荡洗3次后正式取样

#### 底泥采集（老塘必检）
- 彼得森采泥器或长柄铲
- 5点取样混合，取约300g装袋密封

#### 鱼体采样
- 随机撒网或投饵诱集，捞取3-20尾（小规格多取，大规格少取）
- 取背部肌肉（去鳞不去皮）约100g，剪碎
- 如送第三方定量，取肌肉+肝脏+鱼皮各50g分别封装

#### 取样时间
上午8:00-10:00水温较低时。避免投药后5天内取样。

#### 操作步骤

**步骤一：水质检测**
- 取水样至检测瓶刻度线
- 依次滴入pH、氨氮、亚硝酸盐、溶氧检测试剂
- 加盖摇匀，静置显色（各项目显色时间见说明书）
- 与标准比色卡对照读数并记录

**步骤二：鱼体前处理**
- 取鱼肉100g，剪碎至黄豆大小
- 称取匀浆3-5g于离心管（不同检测项目用量不同）
- 加入提取液，剧烈振荡

**步骤三：胶体金卡检测**
- 取提取上清液，加样孔滴入3-4滴
- 水平放置10-15min（孔雀石绿等部分项目需更长显色时间）
- 判读标准与蔬菜类SOP相同

**步骤四：底泥孔雀石绿检测**
- 底泥风干后研磨过筛
- 称取5g+提取液20mL振荡提取
- 取上清按胶体金卡操作

#### 注意事项
- 鱼体采样必须保证鱼体鲜活（濒死鱼代谢物可能偏差）
- 孔雀石绿检测需同时测隐性孔雀石绿（氧化处理后测）
- 硝基呋喃代谢物检测需盐酸衍生化处理后再上卡（按说明书）
- 水质检测试剂有保质期，过期试剂直接废弃不可用
- 夏季高温样品需放入保温箱冰袋运输

### SOP-004 荔枝抽样检测操作规程

#### 适用范围
荔枝（妃子笑、桂味、糯米糍等品种），果实发育期约60-70天。

#### 取样时机
花后第3/7/15/20天、膨大期第25/30/35/40/45天、成熟期第50/55天、采收前3天、采收日、采收后24h。

#### 取样点位
- 果园对角线5棵树法（四角+中心各1棵）
- 大树选东南西北4个方向各取2颗果（每棵树8颗）
- 幼果期果实不足则全树取幼果+嫩叶混合样

#### 取样量
- 速测：5棵树共取40颗，混合后取约500g果肉+果皮剪碎
- 留样：另取20颗，整果装袋密封标记
- 叶片样：每棵树取10片中部功能叶

#### 取样时间
上午7:00-9:00。避开用药后安全间隔期内。雨后24小时内不取样。

#### 操作步骤

**步骤一：果实前处理**
- 带皮样品：用于检测果皮残留（模拟商品状态）
- 去核取果肉+果皮约100g，剪碎
- 称取10g样品于提取瓶中（酶抑制法）或5g（胶体金法）

**步骤二：提取处理**
- 加入配套提取液
- 手动振荡3-5min（荔枝果肉含糖高，提取时间需延长）
- 过滤或静置后取上清液

**步骤三：多联检卡操作**
- 用配套滴管取上清液
- 依次滴入各检测窗口（多联卡通常有4-8个窗口，对应不同农药）
- 按说明书规定时间（通常15-20min）逐窗判读

**步骤四：二氧化硫检测（采后保鲜剂）**
- 取荔枝果肉5g+蒸馏水10mL浸泡10min
- 取浸泡液滴入二氧化硫快速检测管
- 与比色卡对照，颜色越深残留越高
- 限值：≤50mg/kg（按GB 2760）

#### 注意事项
- 荔枝糖分高易招虫，采样时观察果实表面是否有虫蛀孔
- 果皮残留和果肉内部残留检测有差异，建议分别制样
- 高温季节采摘荔枝，送样需全程冷链
- 二氧化硫检测需在采收后24小时内完成（挥发性）
- 山地果园步行难度大，外勤携带设备从简

## 6. 专家评审质疑与完善意见

### 6.1 质疑一：快速检测方法的可靠性

**专家会问**："胶体金卡的假阳性率和假阴性率分别是多少？酶抑制法对某些有机磷不灵敏怎么办？用快检结果做预警甚至暂停出库，依据站得住吗？"

**完善意见**：
1. 明确"快检阳性=筛查阳性，非确证结论"
2. 增加方法验证章节：每种胶体金卡使用前须经第三方做50批次比对验证
3. 建立"快检阳性→酶抑制复核→第三方LC-MS/MS确证"三级判定流程
4. 每批次胶体金卡至少做1次阳性质控+1次阴性质控

### 6.2 质疑二：抽样方案的代表性

**专家会问**："5点取样法对不规则地块适用吗？蔬菜200g样品够不够代表一整块地？生猪抽10%凭什么是10%？"

**完善意见**：
1. 引用国标：GB/T 8855（水果蔬菜取样）、NY/T 789（农残采样）
2. 蔬菜取样改为：≤0.5亩5点，0.5-3亩8-10点，>3亩每增1亩增2点，总样不少于1kg
3. 生猪抽样：补充置信度推算依据
4. 水产抽样：5亩以下≥10尾，5-20亩≥20尾

### 6.3 质疑三：人员资质与培训周期

**专家会问**："3天培训就能上岗？检测人员不需要相关专业背景？"

**完善意见**：
1. 人员门槛：中专以上，农业/食品/化学/兽医相关专业优先
2. 培训延长至5-7天（含2天田间实操）
3. 设立实习期（前10次由老员工带教复核）
4. 建立人员资质档案（培训记录、考核成绩、上岗证号）
5. 与农检中心或高校建立技术合作

### 6.4 质疑四：成本测算偏低

**专家会问**："交通费一趟能跑来回？胶体金卡单价是怎么定价的？设备折旧、社保、保险算了没有？"

**完善意见**：
1. 重算交通成本（保守按实际路况估算）
2. 增加遗漏成本项：社保公积金（30-40%）、商业意外险、设备折旧、办公场地、通讯费、不可预见费（10%）
3. 附耗材采购比价表

### 6.5 质疑五：法律效力与证据链

**专家会问**："快检结果能不能作为保险拒赔的证据？样品留样是否合规？有没有完整的证据链？"

**完善意见**：
1. 现场抽样必须三方签字（抽样员+被检方+见证人）
2. 样品封样用防拆封条，三方签名
3. 留样保存-18℃±2℃
4. 全程拍照（取样点GPS+封样+检测结果）
5. 快检阳性不具有最终判定效力，赔付判定必须基于CMA资质第三方定量报告
6. 被检方有权在3个工作日内申请复检

### 6.6 质疑六：质量控制体系

**专家会问**："自身的质量控制体系在哪里？有没有参加能力验证？设备校准记录在哪？"

**完善意见**：
1. 每日质控：至少1个空白对照+1个阳性质控
2. 外部质控：每半年参加1次省级能力验证，每季度与第三方做10批次比对
3. 设备管理：速测仪每日校准，温湿度计/天平每年检定
4. 胶体金卡入库前抽样验证
5. 制定《质量控制年度计划》作为附件

### 6.7 质疑七：品类方案完整性

**专家会问**："其他品类怎么办？蛋鸡、肉鸡、茶叶、食用菌？荔枝不具有全国推广性。"

**完善意见**：
1. 补充品类选择依据（基于投保结构+历史风险数据）
2. 增加品类扩展路径：第1年四品类→第2年扩展肉鸡/蛋鸡/茶叶→第3年扩展粮油/食用菌
3. 附通用SOP模板（新品类替换指标/取样/频次即可）

### 6.8 质疑八：数据质量与信息化风险

**专家会问**："移动端开发8,000够吗？数据安全考虑过没有？系统崩溃了怎么办？"

**完善意见**：
1. 分档说明：轻量版（现成工具<1万）→标准版（定制5-8万）
2. 数据安全：账号分级、修改留痕、本地每日备份、敏感数据脱敏
3. 灾备方案：系统故障期间启用纸质备份

### 6.9 质疑九：绩效考核实际脱节

**专家会问**："阳性检出加分会不会诱导专挑有问题的采？不同品类工作难度差异怎么体现？"

**完善意见**：
1. 删除"阳性检出加分"，改为"发现系统性风险+5分"
2. 增加"漏检追溯"机制
3. 差异化考核权重：山区/果园外勤加5-10分基础分
4. 增加团队协作互评维度

### 6.10 质疑十：预警联动与保险赔付闭环

**专家会问**："预警发了之后谁跟进处置？处置不力怎么办？确证期间农产品已采收怎么处理？"

**完善意见**：
1. 建立预警闭环管理：一级1个工作日内响应，二级4小时内，三级1小时内
2. 暂停出库机制：二级及以上预警可即时暂停，发放双方签字的《暂停出库通知书》
3. 预警处置台账：每条预警有编号、触发时间、处置动作、闭环时间
4. 保险条款建议写入"投保方须配合事中抽样检测，不配合的可不予赔付"

### 整改优先级

**高优先级（评审前必须完善）**：法律效力与证据链、快检方法可靠性验证、成本测算修正、预警闭环管理

**中优先级（评审时需有准备）**：质量控制体系、抽样代表性、人员资质、数据安全

**低优先级（后续迭代）**：品类扩展计划、绩效考核优化

### 7.4 风险评分引擎

```
风险评分计算逻辑：
1. 品类基础分 (0-30)：绿叶菜30, 瓜果20, 根茎10
2. 历史超标记录 (0-30)：min(违规次数×10, 30)
3. 近期检测结果趋势 (0-20)：恶化+20, 改善+0
4. 用药合规记录 (0-20)：记录不全+15, 有禁用药标志+20

评分→风险等级映射：
0-30: 低风险  |  31-60: 中风险  |  61-100: 高风险
```

### 7.5 预警自动判定规则

```
1. 禁用农药检出 → 三级预警 (自动)
2. 胶体金阳性 + 酶抑制复核阳性 → 二级预警 (自动)
3. 限量值超标 >2倍 → 二级预警
4. 临界值 (限量值的80-100%) → 一级预警
5. 台账记录缺失/造假 → 一级预警
6. 同主体连续2次一级预警 → 升级为二级预警
```


### 7.7 离线采集方案

移动端使用 IndexedDB + Service Worker 实现离线能力：

1. 外勤到达现场，APP自动缓存主体基础信息
2. 拍照→本地压缩（限制1024px宽，单张<200KB）
3. 填表→存入IndexedDB
4. 提交时：有网直接上传，无网标记"待上传"
5. 网络恢复后自动同步，冲突以提交时间戳为准

### 7.8 部署方案（试点期）

单台服务器 Docker Compose：

```
配置: 4核8G, 200G SSD
系统: Ubuntu 22.04
容器: nginx + backend + postgres + redis + minio
```

**第一阶段（1-2个月）MVP**：主体管理+抽样任务手动创建+移动端H5基础表单+手动预警

**第二阶段（3-4个月）自动化**：风险评分引擎+抽样计划自动生成+自动预警+飞书通知

**第三阶段（5-6个月）完善**：离线采集+路线规划+数据看板+第三方报告对接+飞书日历同步

### 8.3 模块一：风险流程引擎

用四张多维表格承载全部业务流程。

**表1：投保主体档案**
- 主体名称（文本）、品类（单选）、地址（位置）、风险等级（AI自动计算）、风险评分（AI自动）、状态（正常/预警中/停业）

**表2：抽样任务**
- 关联主体（关联表1）、计划日期（日期，AI自动生成）、阶段（单选）、检测项（多选）、指派外勤员（人员）、状态（待派/已派/执行中/已完成）

**表3：检测结果**
- 关联任务（关联表2）、检测项目（文本）、方法（单选）、结果（阴性/阳性/无效）、定量值（数字）、限量标准（数字）、采样图片（附件）、GPS定位（位置）、三方签字图片（附件）

**表4：预警记录**
- 关联主体（关联表1）、预警等级（单选，AI自动写入）、触发结果（关联表3）、处置状态（待处理/处理中/已闭环）、处置人（人员）、闭环时间（日期）

**自动化规则**：
- 表3新增阳性记录→自动在表4创建预警+@相关人员
- AI助手定时扫描→自动更新表1风险评分
- 超期任务→自动@外勤员

### 8.4 模块二：数据收集

外勤员现场操作流程：
1. 打开飞书，进入多维表格
2. 找到今日任务
3. 现场检测，手机拍照
4. 填结果、上传照片、GPS自动获取
5. 保存完成

**离线支持**：飞书多维表格支持离线缓存，田间无信号时正常操作，回程有网络自动同步。

### 8.5 模块三：日历与活动管理

**排期逻辑**（AI助手自动执行）：
1. 扫描多维表格主体资料（品类+起始日期）
2. 按SOP节点计算未来一周各主体的抽样日期
3. 结合外勤员日历找空档
4. 同一片区合并路线
5. 在飞书日历中创建日程+@外勤员
6. 同步更新多维表格任务表

**冲突检测**：单日单人任务≤3户，超标自动标红提醒。

### 8.6 AI 助手的自动化能力

**每日自动**：
- 晨间任务推送（今日抽样清单+超期提醒）
- 预警巡检（发现阳性→群内@项目负责人）
- 超期催办

**每周/每月自动**：
- 周报/月报生成
- 风险主体排名
- 排期优化
- 活动提醒（培训/会议前30分钟）

**群内@即用**：
- "@_user_1 生成下周抽样计划"
- "@_user_1 查看张三养殖场全部检测历史"
- "@_user_1 本月预警汇总"
- "@_user_1 新投保主体入库"

### 8.7 落地三步走

**第1步（当天）**：建多维表格模板（4张表+关联关系+自动化规则）

**第2步（本周）**：选1户跑通完整流程（入库→任务→采样→结果→预警→处置→闭环）

**第3步（下周）**：50户全部入库，AI批量生成排期，飞书日历排好，开始日常运转

### 8.8 技术边界说明

**飞书原生方案能覆盖95%的需求。**

飞书暂时做不好的部分（暂不影响使用）：
- 复杂路线优化算法（50户手动排可观）
- 大屏数据可视化（飞书仪表盘能做基础图表）
- 第三方检测报告自动解析（PDF OCR）

等规模上来了再考虑补充。

### 8.9 方案选择建议

首期试点（50户）强烈推荐飞书原生方案。零开发、零运维、零服务器成本，1-2天就上线。飞书的多维表格+群机器人+AI助手已经能覆盖风险流程、数据采集、日历排期三大模块。等试点跑通、规模扩展到200户以上时，再根据实际痛点考虑开发独立系统。
